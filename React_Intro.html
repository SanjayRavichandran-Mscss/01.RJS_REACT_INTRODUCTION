React JS

React JS is a JavaScript Library for building FrontEnd Application or user Interfaces (UI)

React JS allows us to create reusable UI components.

Components are the building blocks of any React app.

It is created by Facebook.

DIRECTORY or FOLDER STRUCTURE OF REACT

node_modules

here , Installed all the modules that is react modules , vite modules and also the dependencies.

public

here , we can place audio files , video files , pictures and etc.., It is display the files in browser without processing the files.

src

src means source , This is very important folder.

In this folder , It first consist of assets , In this asset folder also we can use for store the files ie,. video, image etc.. but it can be processed with the React app and display in the browser.

App.css

The styling of home page of React + Vite.

App.jsx

jsx means JAVASCRIPT WITH XML .

The main file of the React project is App.jsx

here we can modify any code , It reflect in the browser page.

index.css

The index.css is a stylesheet , which is for main.jsx

main.jsx

We can Register All the components in this main.jsx file.

Here , App() is a function in the App.jsx file , This function is call in the main.jsx file.

create new component

We should create a component folder(use folder name as ,"component" or "components" The folder name is Important for react) in the src folder.

In the component folder , we can create the component file as (.jsx) extension , for example (demo.jsx)

Install Extension : [ES7 React/Redux/GraphQL/React-Native snippets]

This Extension is for small keyword to create code snippets.

rafc

In the component file (like Demo.jsx) , Just Type "rafc" and press Enter to create the basic component with import and export statement.

The name of export statement(ie., variable) is similar to the FileName.jsx of component.

One component will return only one Child , But the one Child can be consists n-numbers of child(NESTED CHILD).

Fragment

The Fragment is a builtIn component in React.

The main purpose of the Fragment Tag is to reduce the Extra div tag.

It is used Instead of div tag.

<> (Empty Fragment)

This also the Fragment , we can also use this Empty Fragment instead of Fragment tag.

The both Fragment and Empty Fragment tag are same.
